Empareja las Parejas

Instrucciones

A continuación se detallan las instrucciones del juego:

  • Modo Juego: Inicia desde el Nivel 1 y avanza hasta el Nivel 4. Tienes un número limitado de errores por nivel. Si excedes los errores permitidos, retrocedes un nivel.
  • Modo Libre: Puedes seleccionar cualquier nivel sin restricciones y con intentos infinitos.
  • Arrastra cada elemento sobre su pareja correspondiente.
  • Si emparejas correctamente, ambos elementos desaparecerán.
  • Empareja todos los elementos para completar el nivel.
